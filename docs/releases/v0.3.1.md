## What This Enables

- **Auto-detection**: Chat input like "debug this" or "find bugs" automatically matches and suggests the right skill
- **Full lifecycle coverage**: From ideation (office-hours) through coding (tdd, debugging) to monitoring (canary)
- **Safety controls**: careful/freeze/guard/unfreeze for destructive operation protection
- **Design support**: consultation, shotgun exploration, HTML generation, visual review
- **Quality assurance**: QA with Playwright, security audit (OWASP+STRIDE), performance benchmarking

## Migration

No migration needed. All skills are auto-discovered from `skills/builtin/` at startup.
