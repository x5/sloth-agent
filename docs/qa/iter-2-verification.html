Task 2.2 — 聊天 API + SSE 流式
POST /chat + /chat/stream + GET /messages
19
POST /api/inspirations/{id}/chat — 非流式聊天
发送消息 → Lead Agent 回复。Human Message 和 Agent Message 都存入数据库。
System prompt 从 AgentTemplate 加载。历史消息（最近 1000 条）作为上下文。
Agent 状态：发送中 → working，完成 → idle，错误 → error。
20
POST /api/inspirations/{id}/chat/stream — SSE 流式
text/event-stream 正确。每 token 推送 "data: {token}\n\n"。
流结束推送 "data: [DONE]\n\n"。完整回复在流结束后存入数据库。
Agent 状态正确切换。
21
GET /api/inspirations/{id}/messages — 消息历史
返回历史消息列表。limit 默认 50。cursor 分页（before 参数）。
消息按时间升序。每条消息附带 agent_name / agent_number / agent_model。
22
上下文构建 — 1000 条 + role 映射
human → user, agent → assistant, system → system。
System prompt 在消息数组第一位。
消息取最近 1000 条，时间倒序查再反转。
23
LLM 调用失败 → 错误处理
网络错误 → Agent status 变 "error" → 返回 HTTP 502。
SSE 模式 → "data: {error}\n\n" → "data: [DONE]\n\n"。
Agent 不丢失响应能力（下次消息正常）。
24
无 Lead Agent → 400 错误
Inspiration 没有 Lead Agent 时，POST /chat 返回 400 "No Lead Agent found for this Inspiration"。
